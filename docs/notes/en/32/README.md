# cpu: architectures

* `x86_64`, `x86`, `amd64`: 64-bit *amd/intel*
* `AArch64`, `arm64`, `ARMv8`, `ARMv9`: 64-bit *arm*
* `i386`: 32-bit *amd/intel*
* `AArch32`, `arm`, `ARMv1` to `ARMv7`: 32-bit *arm*
* `rv64gc`, `rv64g`: 64-bit *risc-v*
* `ppc64le`: 64-bit *powerpc*
